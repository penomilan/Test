Test Teorija

  1. Sta je to JSON?
    2. Java Script Object Notation, standard za razmenu podataka koji je ljudima razumljiv.
  2. Sta je to XML?
    3. Format podataka u elektronskoj formi koji se moze koristiti za njihovo skladistenje i razmenu.
  3. Sta je to URL?
    4. Uniform Resorce Locator. Niz karaktera koji se koristi za lociranje nekog sadrzaja internetu.
  4. Sta je to HTTP?
    5. Hyper Text Transfer Protocol. Transfer protokol, metod prenosa podataka na web-u., protokol za komunikaciju izmedju servera i klijenta.
  5. Sta je to RESTful API?
    6. Representational state transfer. Nacin pruzanja operativnosti izmedju kompjuterskih sistema na internetu.
  6. Koje HTTP metode imamo?
    7. Post i Get
  7. Sta je to DNS?
    8. Domain Name System. Sistem imenovanja IP adrese.
  8. Sta je to private IP?
    9. Privatna IP adresa se koristi za lokalne mreze, i uredjaji koji ih koriste ne mogu ucestvovati u internet saobracaju.
  9. Sta je to AJAX?
    10. Asynchronous JavaScript and XML. Grupa tehnologija za razvoj veb aplikacija na klijentskoj strani.
  10. Sta je to TCP/IP?
    11. Skup protokola, koji omogucava umrezenim racunarima da dele medjusobno informacije
  11. Sta je to kes memorija?
    12. Memorija malog kapaciteta u koju se skladiste podaci koji se cesto koriste pri upotrebi uredjaja.
  12. Koja je razlika izmedju klase i objekta?
    13. Objekat je tip klase. Klasa je sablon za instanciranje objekata. Npr, Klasa je ZIVOTINJA, objekti su PAS, MACKA, ZABA.
